# Play Store Assets & ASO Guide

## 📱 App Name
**PG Management System** (formerly PGMS)

## 🎯 App Icon Requirements

### Play Store Icon (Launcher Icon)
- **Size**: 512 x 512 pixels
- **Format**: PNG-32 with transparency
- **Location**: `assets/app-icon.png` (already exists)
- **Safe Zone**: Keep important content within 384 x 384 px center (bleed area is 66px on each side)
- **Shape**: Square with transparency (Android adapts to device shape)

### Adaptive Icon (Android 8.0+)
- **Foreground**: 1080 x 1080 px PNG with transparency
- **Background**: Color or 1080 x 1080 px PNG
- **Location**: `assets/adaptive-logo.png` (already exists)

## 🖼️ Required Screenshots

### Phone Screenshots (Required - Minimum 2, Recommended 5-8)
**Dimensions:**
- 16:9: 1920 x 1080 or 1280 x 720
- 9:16: 1080 x 1920 or 720 x 1280 (Portrait recommended)
- 2:1: 1920 x 960

**Screenshots to Capture:**
1. **Dashboard/Home** - Show the main dashboard with key metrics
2. **Tenant List** - Display tenant management interface
3. **Room Management** - Show room/bed allocation screen
4. **Rent Collection** - Payment tracking interface
5. **Complaint Management** - Issue tracking screen
6. **Notifications** - Push notification examples
7. **Profile/Tenant Details** - Detailed tenant view
8. **Reports/Analytics** - Financial reports screen

**Tips:**
- Remove status bar (use clean screenshots)
- Show real data (not empty states)
- Use consistent device frame
- Highlight key features with captions

### Tablet Screenshots (Optional but Recommended)
**Dimensions:**
- 16:9: 2560 x 1440 or 1920 x 1080
- 9:16: 1440 x 2560 or 1080 x 1920

## 🎨 Feature Graphic (Required)
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPG
- **Safe Zone**: Keep text/logo within 924 x 400 px center
- **Content**: 
  - App logo
  - Tagline: "Manage Your PG Business Effortlessly"
  - Key value proposition
  - Brand colors: #3B82F6 (primary), #0F172A (dark)

## 🎬 Promo Video (Optional but HIGHLY Recommended for ASO)
- **Format**: MP4 or 3GP
- **Max Size**: 100MB
- **Recommended Length**: 30-60 seconds
- **Resolution**: 1280 x 720 or 1920 x 1080
- **Content Structure:**
  1. Hook (0-5s): Problem statement
  2. Features (5-40s): Showcase key features
  3. CTA (40-60s): Download now

## 📝 ASO (App Store Optimization) Checklist

### Title Optimization ✅
- **Current**: "PG Management System"
- **Character Count**: 22/30 (Good - leaves room for keywords)
- **Keywords in Title**: PG, Management, System

### Short Description (80 characters max) ✅
```
Manage PGs, hostels & tenants effortlessly. Rent tracking, payments, complaints & more!
```

### Full Description Keywords Density
Target these high-priority keywords in your description:
1. PG management (3-5 mentions)
2. Hostel management (2-3 mentions)
3. Rent tracking (2-3 mentions)
4. Tenant management (2-3 mentions)
5. Property management (2-3 mentions)

### Backend Keywords (Play Console)
In Play Console, add these in the "Store Listing" > "Manage countries / regions" > "App description" hidden keywords:
```
pgms, paying guest, hostel manager, room rent, tenant tracker, property manager, landlord app, rental management, accommodation, student housing, bed allocation, rent collection, pg owner, hostel owner, room booking, occupancy tracker
```
